# Security Policy

We take security issues seriously and appreciate responsible disclosure.

## Reporting a Vulnerability

If you believe you have found a security issue, please do one of the following:

- Prefer GitHub Security Advisories (if enabled for this repo).
- Otherwise, contact the maintainers via GitHub by opening an issue marked "security" with minimal details and ask for a private channel.

Please avoid posting exploit details publicly until the issue is triaged.

## What to Include

- A clear description of the issue
- Steps to reproduce
- Impact assessment if known
- Any suggested fixes or mitigations
